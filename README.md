# NEato

NES emulator

![Current Status](http://dl.dropbox.com/u/24494398/neato-development/status.png)

## Playable Games

* Super Mario
* Donkey Kong
* Pacman

## Keys

* UP - UP ARROW
* DONW - DOWN ARROW
* LEFT - LEFT ARROW
* RIGHT - RIGHT ARROW
* START - SPACE
* SELECT - ENTER
* A - a
* B - s

## Usage

    $> ./neato super_mario.nes
